# بطه 🦆 — حارس الواحة

لعبة تصويب ثلاثية الأبعاد (FPS) بنظام موجات تعمل في المتصفح: دافع عن الواحة ضد أسراب **البط الآلي**!

- كل الواجهات **عربية RTL**.
- **بدون أي أصول خارجية**: كل المجسمات مبنية من Primitives بالكود، وكل القوام مرسوم عبر Canvas،
  وكل الأصوات مُخلَّقة عبر WebAudio. لا صور، لا موديلات، لا ملفات صوت، لا خطوط محمّلة.
- **بدون دماء**: أعداء روبوتية كرتونية تنفرقع ريشًا وشررًا.

## التشغيل

اللعبة ملفات ثابتة (ES Modules) — تحتاج فقط أي خادم محلي بسيط:

```bash
python3 -m http.server 8765
# ثم افتح:  http://localhost:8765
```

أو: `npx serve .`

> ملاحظة: فتح `index.html` مباشرة من القرص (`file://`) لا يعمل لأن المتصفحات تمنع وحدات ES بدونه خادم.

## التحكم

| المفتاح | الفعل |
|---|---|
| `W A S D` أو الأسهم | الحركة |
| الفأرة | النظر والتصويب |
| زر الفأرة الأيسر | إطلاق (استمر بالضغط للتكرار) |
| `Shift` | جري |
| `Space` | قفز |
| `R` | إعادة تعبئة |
| `Esc` | إيقاف مؤقت |

## بنية المشروع

```
index.html      الصفحة (RTL) وهيكل كل الشاشات
styles.css      أنماط الواجهة العربية
src/            كود اللعبة (ES Modules بدون أي خطوة بناء)
vendor/         مكتبة Three.js مرفقة محليًّا (كود، ترخيص MIT)
docs/SPEC.md    وثيقة المواصفات الكاملة (المراحل ومعايير القبول)
tools/          أداة فحص آلي بلا واجهة (للتطوير فقط)
```

## للتطوير والاختبار الآلي

- `?nolock` في الرابط: تشغيل بدون قفل المؤشر (للفحص الآلي).
- `?autostart` بدء اللعب مباشرة.
- `node tools/shot.mjs` يلتقط لقطات ويرصد أخطاء Console وأي طلب شبكة خارجي.
