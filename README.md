# بطه — شوتر سعودي 3D بالمتصفح

لعبة تصويب ثلاثية الأبعاد **6 ضد 6 ضد بوتات**: أنت + 5 بوتات حلفاء (الفريق الأزرق)
ضد 6 بوتات أعداء (الفريق الأحمر) — على ثلاث خرائط سعودية، بمنظورين FPS/TPS
قابلَين للتبديل أثناء اللعب، وسبعة أسلحة بطابع محلي… بينها **النعال** 🥿.

- كل الواجهات **عربية RTL** بخط IBM Plex Sans Arabic.
- **بدون أي أصول خارجية**: كل المجسمات primitives بالكود، كل القوام Canvas،
  وكل الأصوات والموسيقى Web Audio API — لا موديلات ولا صور ولا ملفات صوت.
- **بدون دماء**: وميض/شرر عند الإصابة، والجثث تتلاشى خلال ثانيتين.

## التشغيل

```bash
npx serve .
# أو: python3 -m http.server 8765
```

ثم افتح العنوان المعروض في المتصفح. (فتح `index.html` مباشرة عبر `file://` لا
يعمل لأن المتصفحات تمنع ES Modules بدون خادم.)

## اللعب

| | |
|---|---|
| **الأنماط** | عدد قتلات (30 قتلة / 10 دقائق، تعادل = موت مفاجئ) · استحواذ (A/B/C، أول 200 نقطة) |
| **المابات** | اشبيليا (الرياض) غروب 🌇 · حارة العوازم (عفيف) ليل 🌙 · الدوادمي ظهيرة ☀️ |
| **الصعوبات** | سهل · وسط · صعب (دقة ورد فعل وسلوك مختلف فعليًا) |
| **الأسلحة** | النمر · ذيب · عين الصقر · أم حصا · زرزور · شدقن (أم خمس) · النعال 🥿 |

### تحكم الدسكتوب

| الفعل | الزر | الفعل | الزر |
|---|---|---|---|
| حركة | WASD | تلقيم | R |
| نظر/تصويب | الماوس + زر أيمن ADS | ضربة نعال سريعة | V |
| إطلاق | زر أيسر | تبديل سلاح/نعال | 1 / 3 أو العجلة |
| سبرنت / قفز | Shift / Space | تبديل منظور FPS/TPS | X |
| قرفصاء / انبطاح | C / Z | سكوربورد | Tab (استمرار) |
| إيقاف مؤقت | Esc أو P | | |

### الجوال

كشف تلقائي: جويستيك يسار للحركة (دفعه كاملًا = سبرنت)، سحب يمين الشاشة للنظر،
وأزرار شفافة (إطلاق، تصويب، قفز، انحناء — نقرة قرفصاء ومطوّلة انبطاح، تلقيم،
🥿، منظور). الشاشة العرضية إلزامية.

## بنية المشروع

```
index.html
styles.css
src/
  main.js              نقطة الدخول وآلة الحالات والريكورد
  engine/              renderer · input (دسكتوب+لمس) · physics · audio · particles · utils
  game/                player (مقاتل+لاعب) · weapons · weapon-models · soldier-model
                       bots · modes (ماتش+نمطان) · hud
  maps/                common (عدّة البناء) · ishbiliya · awazem · dawadmi · (testmap للفحص)
  ui/                  menus · screens
  data/                weapons-data (جدول التوازن) · bot-names · strings-ar
vendor/                Three.js مرفقة محليًا (كود، ترخيص MIT)
docs/SPEC.md           المواصفات الكاملة (= battah-game-prompt.md)
tools/                 فحص آلي بلا واجهة (للتطوير فقط)
```

## للفحص الآلي (تطوير فقط)

معاملات URL: `?nolock` (بدون قفل مؤشر) · `?autostart` · `?map=ID&mode=tdm|dom&diff=easy|mid|hard&weapon=ID`
· `?dummies=N` (أهداف ساكنة) · `?ff=N` (تسريع محاكاة) · `?touch=1` (فرض واجهة اللمس).

```bash
node tools/shot.mjs "?nolock&autostart" out.png   # لقطة + رصد أخطاء/طلبات خارجية
```
